
# Wood围棋规则

Wood围棋规则是一种创新的围棋变体规则，通过白棋首手座子、不贴目、平局白胜等机制实现公平对局。

## 核心规则

1. 空棋盘开局
2. 白棋第一手必须座子：
   - 19×19棋盘：1路或2路
   - 13×13棋盘：仅限1路
   - 9×9棋盘：仅限1路
3. 白座子完成后，黑棋正式先行
4. 之后双方交替落子，正常对局
5. 终局不贴目、不贴子、不扣点
6. 子多/目多者胜，子/目相同时白胜

## 使用说明

1. 下载规则文档
2. 在对局前确定白方座子位置
3. 按照规则进行对局
4. 终局按规则判定胜负
