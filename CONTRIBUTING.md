
# 贡献指南

感谢您对Wood围棋规则的关注！我们欢迎任何形式的贡献和改进意见。

## 如何贡献

### 报告问题
如果您发现规则描述中的错误或不清晰之处，请通过以下方式报告：
1. 在GitHub上创建Issue
2. 详细描述问题所在
3. 提供改进建议（如适用）

### 提交改进
如果您希望改进规则文档：
1. Fork本仓库
2. 创建新分支进行修改
3. 提交Pull Request
4. 等待维护者审核

### 讨论规则
如果您对规则有疑问或建议：
1. 在Issues中创建讨论话题
2. 或通过邮件联系维护者

## 贡献内容

我们接受以下类型的贡献：
- 规则描述的优化
- 语言表达的改进
- 格式和排版的调整
- 示例对局的添加
- 翻译成其他语言

## 注意事项

1. 请保持规则的核心机制不变
2. 任何修改都应有助于提高规则的清晰度和可理解性
3. 避免引入与规则无关的内容
4. 尊重其他贡献者的意见和建议

## 社区行为准则

1. 保持友善和尊重
2. 建设性地提出意见
3. 避免无意义的争论
4. 关注规则本身而非个人偏好

感谢您的贡献！
